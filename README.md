 #### 22.3.18~19일
  - 아이디 찾기 기능을 추가하였으며 이번에도 AJAX를 사용하였다.
  - 처음으로 회원의 이름과 이메일을 작성하면 값을 Mysql로 보내어 Select를하여 값비교를 하였다.

  - 자바에서 연산하여 C:Set을 사용하여 출력되는 아이디를 1차적으로 가려놓았고 다시 핸드폰 인증을하여 아이디 찾는 방식으로 진행하였다.
